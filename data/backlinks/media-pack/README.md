# Voyager Balloons media pack para backlinks

Este paquete sirve para enviar material editorial a medios, guias de viaje, blogs y plataformas internacionales.

## Donde dejar las fotos

Deja las fotos originales aqui:

`data/backlinks/media-pack/photos/raw/`

Despues seleccionaremos las mejores y las pasaremos a:

- `data/backlinks/media-pack/photos/selected/`: seleccion final para enviar.
- `data/backlinks/media-pack/photos/optimized/`: versiones comprimidas/WebP/JPG ligeras para compartir.

Las fotos estan ignoradas por Git para no subir archivos pesados ni material sensible.

## Que fotos necesitamos

Prioridad alta:

- Globo en vuelo con Segovia reconocible.
- Vista del Alcazar desde el aire.
- Vista de la Catedral o casco historico.
- Amanecer con luz bonita.
- Pasajeros felices sin invadir privacidad.
- Brindis con cava despues del vuelo.
- Equipo/piloto en una escena profesional.
- Imagen vertical para medios moviles/redes.
- Imagen horizontal hero para articulos.

Evitar:

- Fotos borrosas.
- Fotos demasiado oscuras.
- Primeros planos de clientes sin permiso.
- Matriculas, datos personales o menores reconocibles sin autorizacion.
- Imagenes con desayuno, porque ya no queremos comunicarlo.

## Uso recomendado

1. Copiar fotos originales en `photos/raw/`.
2. Seleccionar 12-18 imagenes fuertes.
3. Renombrarlas con nombres descriptivos.
4. Optimizar versiones ligeras.
5. Crear una carpeta compartible en Google Drive solo con `selected/` u `optimized/`.
6. Incluir el enlace en los mensajes aprobados.
